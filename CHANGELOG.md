# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-03-23

### Added
- Initial release of YOLOv7-Ultimate
- Complete training pipeline with data preparation tools
- TensorRT deployment support with FP16/INT8 quantization
- Comprehensive documentation (TRAINING.md, DEPLOYMENT.md, FAQ.md)
- Example scripts for inference and training
- Multi-GPU distributed training support
- Model export to ONNX format
- Camera and video inference support

### Features
- YOLOv7 training from scratch and fine-tuning
- Custom dataset support with data preparation utilities
- TensorRT accelerated inference
- FP16 and INT8 precision support
- Batch inference capability
- Real-time camera inference
- Comprehensive evaluation metrics

### Documentation
- Added detailed training guide
- Added deployment guide with TensorRT instructions
- Added FAQ for common issues
- Added contributing guidelines
